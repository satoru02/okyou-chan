## Build
```
clone the repo.
$ git clone https://github.com/satoru02/okyou-chan.git

# Change to the directory.
$ cd okyou-chan

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
